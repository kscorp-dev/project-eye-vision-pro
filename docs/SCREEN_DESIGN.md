## 2. 앱 아이콘

### visionOS 앱 아이콘 사양

visionOS는 **3개 레이어**의 원형 아이콘을 사용합니다. 사용자가 아이콘을 바라보면 시차 효과(parallax)가 발생합니다.

```
┌─────────────────────────────────────────┐
│         visionOS App Icon               │
│         1024 × 1024 px (원형)            │
│                                         │
│  ┌─── Layer 3 (Front) ──────────────┐   │
│  │                                  │   │
│  │      ✦  눈 모양 아이콘            │   │
│  │     (  👁  )                     │   │
│  │      시선 방향 표시 광선           │   │
│  │                                  │   │
│  └──────────────────────────────────┘   │
│                                         │
│  ┌─── Layer 2 (Middle) ─────────────┐   │
│  │                                  │   │
│  │    🦋  나비 가이드 포인트들        │   │
│  │    ✧ ✦ ✧  빛나는 경로 점들        │   │
```

## 3. 런치 스크린

### 사양
- **타입**: visionOS Window (자동 크기)
- **배경**: `.ultraThinMaterial` (글래스 효과)
- **지속 시간**: 1.5초 → 자동 전환

```
╔═══════════════════════════════════════════════╗
║                                               ║
║                                               ║
║                                               ║
║                  (  👁  )                      ║
║                                               ║
║              E y e J o u r n e y              ║
║                                               ║
║          See the World, Train Your Eyes        ║
║                                               ║
║                  ───●────                     ║
║                 (로딩 바)                      ║
║                                               ║
║                                               ║
╚═══════════════════════════════════════════════╝
  ▲ Glass Background (.ultraThinMaterial)

구성요소:
┌──────────────────────────────────────┐
│ 요소          크기/스타일             │
├──────────────────────────────────────┤
│ 앱 로고       80×80pt, 중앙 정렬      │
│ 앱 이름       .largeTitle, .bold      │
│ 서브타이틀    .subheadline, .secondary │
│ 로딩 바       ProgressView, .tint(.blue)│
│ 배경          .ultraThinMaterial       │
│ 모서리        RoundedRectangle(24)     │
└──────────────────────────────────────┘

애니메이션:
  0.0s  로고 fadeIn + scale(0.8→1.0)
  0.5s  텍스트 fadeIn
  1.0s  로딩 바 표시
  1.5s  전체 fadeOut → 다음 화면
```
